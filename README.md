# 🏰 Castle Match

Jogo match-3 estilo Royal Match feito em HTML/CSS/JS puro.

## 🎮 Como jogar

Clique em duas gemas adjacentes para trocar de lugar e combinar 3 ou mais iguais.

## ✨ Recursos

- **Níveis infinitos** com dificuldade progressiva (Normal → Médio → Difícil → Épico → Lendário)
- **Gemas especiais** criadas por combos:
  - 🚀 Foguete — combo de 4 em linha → limpa linha e coluna
  - 💣 Bomba — combo em L ou T → explode área 5×5
  - ✳️ Cruz — combo em T grande → limpa linha + coluna + bordas
  - 🌈 Arco-íris — combo de 5 → elimina todas as gemas de uma cor
- **Sistema de combos** com badges animados
- **Power-ups**: 🔨 Martelo, 🔀 Shuffle, 💣 Bomba
- **2 chances de continuar** ao perder (com penalidade de jogadas)
- **Ranking** com nome salvo no localStorage
- Queda de gemas fluida com física CSS

## 🚀 Jogar

Abra o `index.html` no navegador — sem instalação necessária!

## 🛠️ Tecnologias

HTML · CSS · JavaScript puro (sem frameworks)
